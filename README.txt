MT Pricing System - GitHub Ready

طريقة الرفع على GitHub Pages:
1. فك الضغط.
2. ادخل GitHub واعمل Repository باسم mt-pricing.
3. ارفع الملفات كلها: index.html / manifest.json / service-worker.js / mt-logo.png / icon files.
4. Settings > Pages.
5. Source: Deploy from branch.
6. Branch: main / root.
7. افتح الرابط من الموبايل واضغط Add to Home Screen.

مهم:
- التطبيق مصمم بلون MT الرسمي من اللوجو.
- التعديل على قواعد التسعير يكون من داخل index.html في object اسمه pricingRules.
- أهم خانة للتسعير الدقيق: Units per Sheet.
